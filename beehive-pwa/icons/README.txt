ICONS NEEDED:
============

You need to add these 3 icon files to this folder:

1. icon-192.png (192x192 pixels)
2. icon-512.png (512x512 pixels)  
3. favicon.ico (32x32 pixels)

EASY WAY TO GET ICONS:
===================

1. Go to: https://favicon.io/emoji-favicons/bee
2. Click the bee emoji üêù
3. Download the zip file
4. Extract and rename:
   - android-chrome-192x192.png ‚Üí icon-192.png
   - android-chrome-512x512.png ‚Üí icon-512.png
   - favicon.ico ‚Üí favicon.ico
5. Copy all 3 files to this icons/ folder

The app will work without icons, but they make it look professional!
